Package: Runtime.Attributes
  Пакет управления атрибутами.
  
Remarks:
  Данный пакет используется для реализации механизма хранения мета-информации об объектах.
  Для любого объекта в луа могут быть установлены атрибуты. Данный механизм используется для возможности хранения мета-информации 
  об объекте с целью её последующей проверки.
  
Group: Classes

+ <Runtime.Attributes.Attribute>
  
Group: Functions
  
Function: attribute(...)
  Создание атрибута.
  
Remarks:
  В случае отсутствия аргументов создаёт новый атрибут. В случае наличия параметров создаёт композитный атрибут, установка которого эквивалентна
  установка каждого из атрибутов параметров.
  
Returns:
  Объект-атрибут класса <Runtime.Attribute>
  
Function: attributesof(object)
  Возвращение итератора списка атрибутов указанного объекта.
  
Parameters:
  object - Объект, атрибуты которого запрашиваются.
  
Returns:
  Итератор списка атрибутов.
