Title: Основные события системы ввода

Section: События Window Driver

  Список событий Window Driver:

  + Cursor at x y - x.y - оконные координаты системного курсора (в пикселях) 
  + CursorX axis value - относительное положение системного курсора в окне. value может принимать значение от -1 (левый край окна) до 1 (правый край окна) 
  + CursorY axis value - относительное положение системного курсора в окне. value может принимать значение от -1 (нижний край окна) до 1 (верхний край окна) 
  + CursorX delta value - value - смещение курсора по оси X в пикселях 
  + CursorY delta value - value - смещение курсора по оси Y в пикселях 
  + WheelX delta value - наклон колеса мыши в колонках (возможны дробные значения). 
  + WheelY delta value - поворот колеса мыши в строках (возможны дробные значения). 
  + <Button> down - нажатие кнопки. Button может быть Mouse0 - Mouse2 - основные кнопки мыши, MouseX1, MouseX2 - дополнительные кнопки мыши (боковые), WheelXLeft, WheelXRight - наклон колеса мыши влево/вправо, WheelYUp, WheelYDown - вращение колеса мыши вверх/вниз, кнопки клавиатуры. 
  + <Button> up - отпускание кнопки. Button может быть Mouse0 - Mouse2 - основные кнопки мыши, MouseX1, MouseX2 - дополнительные кнопки мыши (боковые), WheelXLeft, WheelXRight - наклон колеса мыши влево/вправо, WheelYUp, WheelYDown - вращение колеса мыши вверх/вниз, кнопки клавиатуры. 
  + <Button> dblclk - двойной щелчёк кнопки (зависит от системных настроек скорости щелчка). Button может быть Mouse0 - Mouse2 - основные кнопки мыши, MouseX1, MouseX2 - дополнительные кнопки мыши (боковые). 
  + <Button> repeat - событие генерируется при удержании кнопки клавиатуры. Частота возникновения зависит от системных настроек. 
  + Char value - value - символ кнопки клавиатуры. 
  + CharCode value - value - код введенного символа в юникод формате. 
  + Window closed - окно было закрыто.
  
Section: События DirectInput Driver

  Список событий DirectInput Driver:

  + AxisName axis value - положение абсолютной оси. Значение value лежит в пределах от -1 до 1. 
  + AxisName delta value - изменение значения относительной оси. Для мыши предопределены имена осей Mouse X Axis, Mouse Y Axis, Mouse Z Axis. 
  + ButtonName down - нажатие кнопки. Приходит от кнопок и POV-переключателей. Для мыши предопределены имена кнопок Mouse Button 1 - Mouse Button 8. 
  + ButtonName down - отпускание кнопки. Приходит от кнопок и POV-переключателей. Для мыши предопределены имена кнопок Mouse Button 1 - Mouse Button 8. 
  + POVName pov angle - угол нажатия POV-переключателия. Angle - значение угла в градусах * 100 (положение вверх - 0, положение влево - 27000). 
  + Device lost - устройство было потеряно (отсоединено). 
  + Device reacquired - устройство снова заработало (присоединено). 
  + Device bad - не удаётся получить доступ к устройству.
